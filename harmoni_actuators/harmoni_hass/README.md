## Setting up your home assistant


 
### Set up credentials
The parameter *credential_path* refers to the file where the home assistant authentication token is stored.

The structure of this json file is: 
```
{
    "token" : "YOUR_HOME_ASSISTANT_TOKEN"
}
```
